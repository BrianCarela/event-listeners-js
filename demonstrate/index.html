Event Listeners!

HTML DOM allows JavaScript to react to HTML events. This means that JavaScript can be executed when an Event (A user interacting with the front of a webpage) occurs, and an Event Listener is how you set up such script to be executed, and what Event should trigger the script to occur.

  1. 1. An Element is selected to listen to an Event


  2. 2. An Event is something the user does to trigger a script. This can be done by clicking an element, hovering over it, and even by pressing something on the keyboard


  3. 3. Once the Event is "heard", the scripts will be executed, which can be anything. In this case, it's making new text appear on the page!
